Hitokoto-Spider
一言爬虫

这是一个能够帮助你抓取一言库的爬虫，将抓取到的文件保存为csv格式的文件，另有可选保存项、断点续抓等功能!

  • Github

Netease-Comment-Spider
网易云音乐评论爬虫

我们经常在网易云能看到一些精彩的评论，但是又想把它保存下来，这时候你就可能需要用到此爬虫保存啦！

  • Github

Minecraft-Skin-Link
Minecraft皮肤真实链接获取器

自从Mojang改了皮肤获取策略后，你是不是在苦恼获取不到正版玩家的皮肤？那你就需要这个程序啦！

  • Github

MCDR-WikiSearcher
MCDR服务器局内搜索

如果你用了MCDReforged作为服务器的启动器，那么你可以使用本插件为你的服务器添加MC百科搜索功能，便民便己！

  • Github

SimpleOP
MCDR局内OP获取/服务器管理

想重启/关闭服务器，但是没有OP权限？没关系，这个插件可以帮助你局内获取OP权限，也可以直接对服务器重启关闭

  • Github

Minecraft-Touhou-Music
MC打开东方BGM的正确方式

使用此材(yin)质(yue)包将你的MC音乐替换为东方BGM，在MC里体会东方神曲的魅力吧！
